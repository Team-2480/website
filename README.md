# Team 2480

Our team website.

## How to install

### Prerequisites
- The git repo cloned in a directory
- [NPM, and Node.js installed](https://nodejs.org/)

### Install

In the repo directory install pnpm:
```
npm install -g pnpm
```

Install the required packages:
```
pnpm install
```

Then run the dev server to see your changes!
```
pnpm run dev
```
